Este proyecto muestra un análisis de mantenibilidad en .NET (C#) utilizando SonarQube en local. El código analizado es una aplicación de consola diseñada intencionadamente con problemas de mantenibilidad.

El análisis se ejecutó con dos Quality Gates distintos:

Quality Gate genérico → resultado Passed

Quality Gate de mantenibilidad estricto → resultado Failed

El código no se modificó entre análisis; el cambio en el resultado se debe únicamente a la configuración del Quality Gate.
