<h1 align="center" style="font-family: Arial; font-size: 42px;">
 SonarQube Python Quality Project
</h1>

<p align="center" style="font-size:18px;">
Proyecto demostrativo de integración y análisis de calidad de código utilizando <b>SonarQube</b> , <b>.NET</b> y <b>Python</b>.
</p>

<hr>

<h2 style="font-size:28px;"> Introducción</h2>

<p style="font-size:16px; line-height:1.6;">
Este repositorio contiene pruebas prácticas de análisis estático, control de calidad y validación de métricas utilizando SonarQube y SonarQube Cloud.
</p>

<p style="font-size:16px; line-height:1.6;">
El objetivo principal es demostrar:
</p>

<ul style="font-size:16px;">
  <li> Configuración del entorno SonarQube</li>
  <li> Ejecución de análisis de calidad</li>
  <li> Gestión de Quality Gates</li>
  <li> Cobertura de pruebas</li>
  <li> Manejo de errores y métricas</li>
</ul>

<hr>

<h2 style="font-size:28px;"> Estructura del Proyecto</h2>

<p style="font-size:16px;">
A continuación encontrarás diferentes carpetas organizadas por lenguaje.
</p>

<p style="font-size:18px;">
 <b>Para revisar las pruebas y configuraciones, entra en cada carpeta.</b>
</p>

<ul style="font-size:16px;">
  <li><b>/pruebaSonarqube.NET</b> → Código base del proyecto .NET.</li>
  <li><b>/images</b> → Evidencias visuales del análisis (capturas de SonarQube).</li>
</ul>

<ul style="font-size:16px;">
  <li><b>/pruebaSonarqubePython</b> → Código base del proyecto Python.</li>
  <li><b>/images</b> → Evidencias visuales del análisis (capturas de SonarQube).</li>
  <li><b>/config</b> → Archivos de configuración como <code>sonar-project.properties</code>.</li>
</ul>
